# 🚗 Car Sales Performance & Trend Analysis Dashboard

## 📊 Project Overview
This project is an interactive dashboard built using Power BI to analyze car sales performance and trends. It provides insights into key metrics such as total sales, average price, and number of cars sold.

---

## 🎯 Objectives
- Analyze overall car sales performance
- Identify sales trends over time
- Understand regional sales distribution
- Compare company-wise performance
- Explore customer and product-level insights

---

## 📈 Key Features
- YTD Total Sales, Avg Price, and Cars Sold KPIs
- Weekly Sales Trend Analysis
- Sales by Body Style and Color
- Dealer Region-wise Sales (Map Visualization)
- Company-wise Sales Performance Table
- Interactive Filters (Body Style, Dealer, Transmission, Engine)

---

## 🛠 Tools & Technologies
- Power BI
- Excel (Dataset)
